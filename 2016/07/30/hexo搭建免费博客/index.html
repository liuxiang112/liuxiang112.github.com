min版的hexo开发模式

/ 安装node 确保正确安装

/ 进入系统盘 建立一个文件 我们叫做demo

/ cd demo

/ 执行 npm install -g hexo

/ 执行 hexo init

/ 执行 hexo g

/ 执行 hexo s

/ 以上步骤确保执行 项目就可以跑起来/
